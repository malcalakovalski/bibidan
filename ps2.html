First, define the thresholds.

binary_wdi <- wdi %>% mutate(corrupt = if_else(corruption > quantile(corruption, 0.2, na.rm = TRUE), "Corrupt", "Not corrupt"), 
    unequal = if_else(gini > 40, "Highly unequal", "Not highly unequal"), country, .keep = "used")

We can get the contingencies in two steps:

binary_wdi %>% drop_na(unequal, corrupt) %>% filter(corrupt == "Corrupt") %>% count(unequal) %>% mutate(prop = n/sum(n, na.rm = TRUE)) %>% 
    gt()
